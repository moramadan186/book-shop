# Books Shop App

A simple website to find the most popular books by browsing pages, discovering book details, and downloading it Ebook.

## Deployment

To run this project, In terminal run

```bash
  npm install
```

```bash
  npm start
```

## Tech Stack

React, TailwindCss, Axios, Swiper, heroicons
